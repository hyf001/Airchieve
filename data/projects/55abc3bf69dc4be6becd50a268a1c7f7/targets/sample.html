项目说明文档

欢迎使用文档处理系统。这是一个示例 Markdown 文件，用于展示 HTML 生成功能。

功能特点

  1. 格式转换：支持多种文档格式互转。
  2. 样式增强：自动应用现代化的 CSS 样式。
  3. 响应式设计：完美适配手机、平板和桌面端。
  4. 交互体验：内置深色模式切换等功能。

使用说明

请将您的源文件放入 assets/ 文件夹，系统将自动识别并处理。您可以指定输出目录和特定的样式偏好。
