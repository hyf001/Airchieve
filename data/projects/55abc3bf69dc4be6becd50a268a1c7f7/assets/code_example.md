# 代码示例演示

本页面展示了文档处理系统的代码高亮功能。

## Python 示例

```python
def hello_world():
    print("Hello, AIrchieve!")

if __name__ == "__main__":
    hello_world()
```

## JavaScript 示例

```javascript
const toggleTheme = () => {
    const body = document.documentElement;
    const isDark = body.getAttribute('data-theme') === 'dark';
    body.setAttribute('data-theme', isDark ? 'light' : 'dark');
};
```

}
